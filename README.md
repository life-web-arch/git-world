# Git World 🚀

Welcome to Git World, a 3D multiplayer visualization of the GitHub universe, inspired by projects like Git City.

This project turns GitHub profiles into a navigable 3D world where every developer gets their own "base." The dimensions of the base are determined by their public and private contributions and repository count.

![image](https://i.imgur.com/your-image-url.png) <!-- You can add a screenshot URL here later -->

## ✨ Features

- **Real-time Multiplayer:** See other developers exploring the world with you.
- **Dynamic Buildings:** Building height is based on total contributions, and width is based on repository count.
- **GitHub Avatars:** Developer profile pictures are displayed on their bases.
- **Fly Mode:** Freely explore the world from the skies.
- **Cross-Platform:** Playable on desktop (WASD + Mouse) and mobile (Joystick).
- **$0 Budget:** Built entirely on a free-tier stack (Next.js, Supabase, Vercel).

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **3D Engine:** React Three Fiber & Drei
- **Physics:** Rapier
- **Multiplayer:** Supabase Realtime
- **Authentication:** NextAuth.js
- **Styling:** Tailwind CSS



Open [https://git-world-sigma.vercel.app](https://git-world-sigma.vercel.app) with your browser to see the result.
